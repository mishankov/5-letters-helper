# 5 letters helper

## Команды

```shell
# Подготовка 5-ти буквенных существительных
go run ./cmd/prepareWords

# Подготовка БД
go run ./cmd/prepareDb

# Запуск консольного приложения
go run ./cmd/cli

# Запуск бота
set BOT_SECRET=<token> && go run ./cmd/bot
```
